## Post Body
I just launched a production-ready skin cancer detection portfolio project that combines:

- **EfficientNetB3** transfer learning for 7-class HAM10000 classification
- **FastAPI** backend with JWT authentication and MongoDB prediction history
- **ONNX Runtime** for fast, scalable inference
- **Real Grad-CAM explainability** for visual trust in medical predictions
- **React** frontend with upload flow, history page, and PDF report generation
- **Docker + GitHub Actions** for clean deployment and CI/CD readiness
